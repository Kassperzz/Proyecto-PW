Cambio de Aceite

Diagnóstico

El vehiculo marca DODGE necesitaba un cambio de aceite ya que no se le ha realizado uno desde la ultima vez que se reviso

Mecanico y fecha en el que se realizo

El mecánico Jorge fue el encargado de realizar este trabajo el dia 27/06/2022

el listado de materiales que se utilizaron

Los materiales utilizados son:

  • Embudo (algunas botellas de aceite lo traen incorporado)
  • Un recipiente vacío para recoger el aceite usado.
  • Kit de llaves de acuerdo al modelo de tu carro.
  • Un filtro de aceite nuevo.
  • Gato hidráulico o elevador.
  • Linterna de cabeza o mano.

    • Todos los trabajos realizados al menos poseen una foto.

    Taller Mecanico
    Paso N°1

    Como primer paso se debe retirar el aceite que ya trae el vehiculo para que haci el filtro quede limpio.

    Paso N°2

    Luego de haber retirado el aceite se comienza a introducir el aceite nuevo.

    Paso N°3

    Para finalizar el trabajo se revisan los niveles de aceite para poder culminar con el trabajo y el cliente se vaya feliz.
